# Connect Four

A CLI implementation of the game Connect Four

Created for The Odin Project curriculum

## Setup

Prerequisites

- Ruby 3.2.2

Clone the repository

```bash
git clone git@github.com:hdito/connect-four-ruby.git
```

Install dependencies

```bash
bundle install
```

Run the game

```bash
ruby main.rb
```

Run tests

```bash
rspec
```
